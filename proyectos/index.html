“Árboles Ciudadanos” (2022 - 2023)

“Árboles Ciudadanos”: Composición sonora generativa para proyecto de relatos sonoros inspirados en distintas especies de árboles. (2022 - 2023) ![Image 8] Descripción del trabajo realizado: En este proyecto desarrollé un sistema de composición sonora generativa inspirado en distintas especies arbóreas, como parte de un ejercicio de sonificación de estímulos capturados desde el entorno de cada árbol.
